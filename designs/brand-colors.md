# Applify Brand Colors

## Primary Palette

| Name | Hex | Usage |
|---|---|---|
| Deep Teal | `#0F6E56` | Headings, sidebar, primary structure, logo background |
| Medium Teal | `#1D9E75` | Active states, success indicators |
| Light Teal | `#E1F5EE` | Teal backgrounds, active nav items, tag backgrounds |
| Soft Teal | `#9FE1CB` | Borders, dividers, subtle accents |
| Coral | `#D85A30` | CTA buttons, "ify" wordmark, right leg of logo A |
| Light Coral | `#FAECE7` | Coral tag backgrounds, gap indicators |
| Amber | `#EF9F27` | Fit scores, crossbar of logo A, badges, step numbers |
| Light Amber | `#FAEEDA` | Amber backgrounds, nudge banners |

## Neutral Palette

| Name | Hex | Usage |
|---|---|---|
| Background | `#FAF9F7` | Main app background |
| Card | `#fff` | Card and panel backgrounds |
| Surface | `#F8F7F4` | Subtle surface backgrounds |
| Border | `#E8E6E0` | All borders and dividers |
| Text Primary | `#2C2C2A` | Body text |
| Text Secondary | `#5F5E5A` | Secondary text, descriptions |
| Text Muted | `#888780` | Placeholder text, labels, timestamps |
| Text Faint | `#B4B2A9` | Very muted text, inactive items |

## Logo Colors

- Left leg of A: `#E1F5EE` (cream/light teal) on `#0F6E56` background
- Right leg of A: `#D85A30` (coral)
- Crossbar of A: `#EF9F27` (amber)
- "Appl" wordmark: `#0F6E56` (deep teal)
- "ify" wordmark: `#D85A30` (coral)
- Accent dot between "Appl" and "ify": `#EF9F27` (amber)

## Typography

| Usage | Font | Weight | Size |
|---|---|---|---|
| Hero headings | Georgia, serif | 500 | 38px |
| Section headings | Georgia, serif | 500 | 22-28px |
| Card titles | Georgia, serif | 500 | 16-20px |
| Body text | System sans-serif | 400 | 14-15px |
| Small text | System sans-serif | 400 | 12-13px |
| Labels | System sans-serif | 500 | 10-11px, letter-spaced |
